# Agentic AI - Prompting Effective LLM Reasoning and Planning

This folder host lecture notes and codes from the first Udacity **Agentic AI**  courses - **Prompting for effective LLM Reasoning and Planning**.

There are codes respective to the following topics as well as the final project **Travel Planner Agent**:

1. Role-Based Prompting with Python
2. Chain-of-Thought and ReACT Prompting
3. Prompt Instruction Refinement
4. Chaining Prompts for Agentic Reasoning
5. LLM Feedback Loops

In the notebooks, the section numbers correspond to the section of the Udacity course.
